Research Nexus





Nexus is a tool that helps you read papers and take notes
faster with the help of AI. It covers various domains,
including research papers, legal documents, user manuals,
financial documents, books, and course materials1.
You can explore papers, ask questions, and even start group
chats with multiple papers simultaneously.

Paper with code





This platform provides the latest research papers
along with associated code implementations. It covers
various machine learning domains, such as computer vision,
natural language processing, audio processing, multimodal
learning, and reinforcement learning2. You’ll find rankings,
ratings, and links to both papers and code for each topic.

arXiv




The arXiv preprint server hosts a vast collection of AI
research papers. You can explore recent entries in the artificial
intelligence category, covering topics like
distributed computing, logic, and software engineering3.

Elicit




Elicit is an AI research assistant that allows
you to search, summarize, extract data from, and
chat with over 125 million papers. It’s
used by researchers in academia and industry4.
